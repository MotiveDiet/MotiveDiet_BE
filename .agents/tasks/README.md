# .agents/tasks — 휘발성 작업 상태 (지금은 비어 있음)

## 무슨 용도인가

**사이클이 도는 동안의 임시 상태**를 두는 곳이다. 장기 보관용이 아니다.

원래 설계(rvs-apps)에서는 이슈 단위로 `.workspace/issue-{N}/` 같은 작업 디렉토리를 만들어
진행 중 산출물을 두고, 사이클 마지막 **SWEEP 단계가 청소**했다:

```
1. eslint --fix (미사용 import 제거)
2. 임시 주석 패턴 보고 (자동 삭제는 안 함)
3. 빈 placeholder 정리          ← 여기가 tasks/ 를 치우는 단계
4. 핑퐁 2회 이상이면 learnings.md 에 append
5. cost.json 작성
```

## 왜 지금 비어 있나

**청소해줄 SWEEP 단계가 없다.** 우리는 레이아웃만 가져왔고 사이클 자동화(SWEEP·
learnings.md·cost.json)는 도입하지 않았다. 치우는 주체 없이 임시 파일을 쌓으면
영원히 안 지워진다 — 그게 정확히 이 디렉토리가 막으려던 문제다.

임시 파일이 필요하면 지금은 **git 이 추적하지 않는 스크래치 경로**를 쓰고 있다.

## 언제 채우게 되나

사이클 자동화를 도입할 때. 즉 **"누가 언제 이걸 지우는가"에 답이 생겼을 때** 만든다.
그 답 없이 만들면 쓰레기통이 된다.
