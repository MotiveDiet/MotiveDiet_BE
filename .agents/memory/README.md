# .agents/memory — 인덱스 전용 (지금은 비어 있음)

## 무슨 용도인가

**외부 시스템에 있는 지식의 포인터만** 두는 곳이다. 원본 문서를 여기에 복사하지 않는다.

원래 설계(rvs-apps)에서는 사내 Redmine 일감 번호, 사내 위키 링크 같은 **레포 밖에 사는
자료**의 참조 메타데이터를 모아뒀다. 본문을 복사하면 원본이 바뀔 때 조용히 드리프트하므로,
"어디에 무엇이 있다"는 인덱스만 유지하는 게 핵심이다.

## 왜 지금 비어 있나

**우리는 가리킬 외부 시스템이 없다.** 제품 문서(PRD·API·ROADMAP)가 전부 이 레포의
`docs/` 안에 있고, `AGENTS.md` 가 이미 그 지도를 갖고 있다. 레포 안 문서를 여기서 또
가리키면 포인터의 포인터가 될 뿐이다.

## 언제 채우게 되나

아래 중 하나라도 생기면 그때 만든다:

- 이슈 트래커(Jira/Redmine/GitHub Issues)를 본격적으로 쓰기 시작할 때 — 일감 번호 ↔ 결정 인덱스
- 레포 밖 디자인 원본(Figma 파일 키·노드 ID)을 반복 참조하게 될 때
- 사내 위키·노션처럼 레포에 못 넣는 자료가 생길 때

**주의**: 채울 때도 본문을 복사하지 말 것. 링크와 한 줄 요약까지만.
